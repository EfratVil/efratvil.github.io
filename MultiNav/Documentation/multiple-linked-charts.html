Chapter 9 Multiple Linked Charts

Example of linking multiple scatter plot charts to the same linked line chart. The multiple link is set by setting the same ‘link_id’ paramter in both scatter plots. Note:
Future versions will highlight in all the linked scatter plot, the point that represents the id of the point in current focus.

First let’s create some data:

library(MultiNav)
data <- DendrometerSensors
uni_matrix<-Calc_uni_matrix(as.data.frame(data))

quantiles_matrix<-Calc_quantiles_matrix(as.data.frame(data))

data<-cbind(seq_id=as.numeric(row.names(data)), data)
MultiNav(uni_matrix,type = "scatter_and_linked_line",
         "min", "max", raw_data = data, link_id="aaa",q_data = quantiles_matrix)



MultiNav(uni_matrix,type = "scatter","quantile_25", "quantile_75" , link_id="aaa")
