MultiNav - Multivariate Explanatory Data Analysis

Chapter 11 Example IV - Anomaly Simulation

This is a simulated dataset created with multivariate normal distribution and high correlation between variables. The goal is to test and demonstrate MultiNav capabilities. For this purpus, anomalies with spesific charectaristics were added to the dataset.


## Creating the simulated dataset

Loading the necessary packages:

library(MultiNav)
library(MASS)
library(robustbase)
library(huge)
#library(corrplot)
library(data.table)

We start by creating the simulated dataset,

set.seed(12345)
samples<-200 
r<-0.8
data <- mvrnorm(samples, mu = c(0,0,0), Sigma = matrix(c(1,r,r,
                                                         r,1,r,
                                                         r,r,1), ncol = 3), empirical = TRUE)
colnames(data) <- c(1,2,3)

row.names(data)<-seq(1:dim(data)[1])


Adding several observations with known anomalies.

data[45,]<-data[45,]+c(3, 3, 3 )
data[66,]<-data[66,]+c(-3, -3, -3 )
data[83,]<-data[83,]+c(2, -3, 4 )
data[113,]<-data[113,]+c(2, 2, -2 )
data[137,]<-data[137,]+c(-2, 2, -2 )
data[162,]<-data[137,]+c(1, 1, 0 )
