8.1 Example 1 - Breast Cancer Data Set

Breast Cancer Wisconsin (Diagnostic) The features of the breast-cancer dataset are extracted from medical images of a fine needle aspirate (FNA) describing the cell nuclei. The task of the UCI dataset is to separate cancer from healthy patients. From the 212 malignant instances, 10 were kept as anomalies. This results in a unsupervised anomaly detection dataset containing 367 instances in total and having 2.72% anomalies.

From the paper - A Comparative Evaluation of Unsupervised Anomaly Detection Algorithms for Multivariate Data (http://journals.plos.org/plosone/article?id=10.1371/journal.pone.0152173)

Note - The paper referes to datasets that can be used for benchmarking unsupervised anomaly detection algorithms.

8.6 Examin Correlations

Pearson correlation between variables

cor_est1 <- cor(t(ds),use="pairwise.complete.obs")
corrplot(cor_est1, method = "color")



MCD correlation between variables

rob<- cov.rob(t(ds), cor = TRUE, method = "mcd")
cor_est2 <- rob$cor
corrplot(cor_est2, method = "color")



Croux-Ollerer correlation between variables

cov_est <- spearman.transformed(t(ds),method="npd") 
cor_est3<-cov2cor(cov_est)
corrplot(cor_est3, method = "color")



Pearson correlation between subjects

res_cor <- cor(ds,use="pairwise.complete.obs")
corrplot(res_cor, method = "color", order="hclust", tl.pos="n")
