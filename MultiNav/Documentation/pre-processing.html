MultiNav - Multivariate Explanatory Data Analysis

3 Pre-Processing

The package includes several functions for data pre-processing of common EDA tasks. The output from the function can be used as input for the charts in the package.

3.1 Univariate Matrix

Scope: The univariate matrix creates a collection of univariate descriptive statistics performed on the values of each variable.

Input: The function expects input of a data.frame with continues variables from the same scale. A numeric unique id for each observation can also be included in the data.frame. It should be stored in a field called ‘seq_id’ or ‘n_id’. If the id field does not exist, the row names will be used as the id. Any missing data should be handled before calling the function (for example by interpolation). Note: Any records containing missing values will be omitted by default.

Statistic Description
min Minimum value.
quantile_25 Quantile 25.
median Median value.
quantile_75 Quantile 75.
max Maximum value.
mean Mean value.
sd Standard deviation.
mad Mad - Median Absolute Deviation from the median. Robust estimate of standard deviation.
Skewness A measure of the asymmetry.
Kurtosis A measure of the “tailedness”.


Example

library(MultiNav)
data <- sensors1 
uni_matrix<-Calc_uni_matrix(data)

Arguments:
op - Creates additional set of variables which are the box-cox transformation for each one of the uni_matrix fields to normality. For example for the calculated vector of minimum values called ‘min’ in the uni_matrix table, a new vector will be created called ‘min_bc’. Also another set of variables are created which indicate outlier points (1 - outlier, 0 - normal). The outlier points are marked based on 3 sigma spc rule, of the box-cox transformation. For example the outlier points of the ‘min’ vector will be calculated and stored in the ‘min_op’ field.

uni_matrix<-Calc_uni_matrix(data, op=T)

3.2 Quantiles Matrix

Scope: The quantiles matrix creates a collection of several descriptive statistics performed on each observation over the values of all variables (all variables are on the same scale).

Input: The function expects input of a data.frame with continues variables from the same scale. A numeric unique id for each observation can also be included in the data.frame. It should be stored in a field called ‘seq_id’ or ‘n_id’. If the id field does not exist, the row names will be used as the id. Any missing data should be handled before calling the function (for example by interpolation). Note: Any records containing missing values will be omitted by default.

First Header Second Header
quantile_02 Quantile 02.
quantile_25 Quantile 25.
median Median value.
quantile_75 Quantile 75.
quantile_98 Quantile 98.

Example

quantiles_matrix<-Calc_quantiles_matrix(as.data.frame(data))
