MultiNav - Multivariate Explanatory Data Analysis

4 Basic Charts

MultiNav also provides several basic charts that are used as building blocks for the interactive charts, but can also be used as stand alone.

First let’s create some data:

library(MultiNav)
library(MASS)
set.seed(1234)
n <- 300
Sigma <- matrix(c(1.0, .80, .50, .20,
                  .80, 1.0, .05, .05,
                  .50, .05, 1.0, .05,
                  .20, .05, .05, 1.0), ncol = 4)
data <- data.frame(mvrnorm(n, Sigma, mu = c(100,100,100,100), empirical = TRUE))

4.1 Line Chart

Simple line chart.

MultiNav(data,type = “line”, x= NULL, y= NULL)

Arguments:
data - Data set in data.frame frormat to use for plot.
x - variable name of x values.
y - variable name for y values.

Note:
Future versions will alow horizontal zoom.

Example: 

data<- cbind(seq_id=row.names(data), data)
MultiNav(data,type = "line","seq_id", "X1")

4.2 Scatter Plot

Simple scatter plot chart.

MultiNav(data,type = “scatter”, x= NULL, y= NULL)

Arguments:
data - Data set in data.frame frormat to use for plot.
x - variable name of x values.
y - variable name for y values.

Note:
Future versions will alow zoom.

Example: 

MultiNav(data,"scatter","X1", "X2")

4.3 Network Chart

Network chart, with force layout. It can be used to show the relationship between variables based on a distance matrix. It can be an effective way to show correlation matrix (transformed to distance matrix) in highly correlated data set to show outliers or to show clusters.

MultiNav(data,type = “network”)

Arguments:
data - Data set in data.frame format with expected columns named: “from”, “to”, “weight”. The weight should be scaled to the range 0-300.

Example: 

library(igraph)
data <- sensors1 
cor<-cor(data,use="pairwise.complete.obs")

g  <- graph.adjacency((1-cor)*150,weighted=TRUE, diag=FALSE,mode= "upper")
network <- get.data.frame(g)

head(network)
